# 현금 식대 정액 지급

## 대표 지급구조

실제 식사비 증빙과 무관하게 일정 대상자에게 매월 정액의 식대를 현금으로 지급한다.

## 판정 전 확인할 사실

- 사내급식·식권·현물 식사를 함께 제공하는지
- 전 근로자 또는 일정 직군에 정기 지급하는지
- 결근·휴직 시 감액 여부
- 월 20만원 초과 여부와 비과세 코드

## 영역별 판정

| 영역 | 기본 결론 | 확인 포인트 |
|---|---|---|
| 임금성 | 포함 | 실제 비용정산이 아닌 정기적 근로조건 |
| 통상임금 | 포함 가능 | 정기·일률적이고 소정근로 대가인지 |
| 평균임금 | 포함 | 임금성이 인정된 지급액 |
| 최저임금 | 원칙적 포함 | 현행 산입범위와 월 지급액 |
| 근로소득세 | 월 20만원 이하 비과세 가능 | 식사를 제공받지 않는 근로자 |
| 국민연금 | 과세·비과세 부분 분리 | 비과세 근로소득 제외 원칙 |
| 건강·장기요양 | 과세·비과세 부분 분리 | 시행령상 예외 확인 |
| 고용·산재 | 과세·비과세 부분 분리 | 보수 제외 규정 확인 |

## 결론 변경 조건

영수증에 따라 실제 식사비만 정산하거나 사내급식을 제공하면서 현금 식대를 중복 지급하면 노동법·세법 결론이 달라질 수 있다.

## 필수 증빙

취업규칙, 급여규정, 구내식당·식권 제공내역, 급여대장, 비과세 코드, 월별 한도 대사표.

## 결론

정액 현금 식대는 노동법상 임금에 포함될 가능성이 높지만 세법상으로는 식사를 제공받지 않는 근로자에게 지급한 월 20만원 이하만 비과세한다. 사회보험은 과세·비과세 구간과 각 법령의 예외를 분리한다. ^fact-conclusion
