# 기본급과 법정수당이 구분되지 않은 정액급제

## Facts

근로계약은 기본급과 연장·야간·휴일수당을 구분하지 않고 월 300만 원만 지급한다고 정한다. 계약과 명세서만으로 각 법정수당 금액을 바로 알 수 없다. ^claim-fact-fixed-sum

## Rule Application

먼저 포괄임금약정의 성립과 포함 수당을 확정한다. 성립하더라도 총액에서 약정 법정수당 부분을 가산율에 맞추어 역산하고, 남은 최저임금 비교대상 임금을 소정근로시간으로 나누어 해당 연도 최저임금과 비교한다. ^claim-fact-fixed-sum-application

## Conclusion

정액급 총액이 최저임금과 법정수당을 모두 충족하는지 단계별로 계산하며, 약정 연장시간을 소정근로시간에 합산하지 않는다.
