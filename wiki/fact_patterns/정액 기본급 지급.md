# 정액 기본급 지급

## 대표 지급구조

근로계약과 임금표에 월 기본급이 정해져 있고 소정근로를 제공하면 매월 확정액을 지급한다.

## 판정 전 확인할 사실

- 기본급이 보상하는 소정근로시간과 근무형태
- 결근·휴직·중도 입퇴사 시 일할계산 방식
- 기본급에 법정수당을 섞어 적었는지
- 급여대장·근로계약·회계전표의 금액 일치 여부

## 영역별 판정

| 영역 | 기본 결론 | 확인 포인트 |
|---|---|---|
| 임금성 | 포함 | 소정근로의 직접 대가 |
| 통상임금 | 포함 | 정기·일률적으로 미리 확정된 기본급 |
| 평균임금 | 포함 | 산정사유 발생 전 3개월 지급액 |
| 최저임금 | 원칙적 포함 | 비교대상 임금과 소정근로시간을 함께 확정 |
| 근로소득세 | 과세 | 별도 비과세 항목이 아님 |
| 국민연금 | 산입 | 근로소득에서 제외 사유 없음 |
| 건강·장기요양 | 산입 | 근로 대가 보수 |
| 고용·산재 | 산입 | 근로소득 기초 보수 |

## 결론 변경 조건

기본급 명목 안에 연장·야간·휴일수당을 포함했다면 정액급 자체와 법정수당 구분, 약정 시간, 실제 근로시간을 다시 판정한다.

## 필수 증빙

근로계약서, 임금표, 급여대장, 근태자료, 일할계산 규정, 법정수당 산출표.

## 결론

순수한 월 정액 기본급은 노동법상 임금·통상임금·평균임금과 각 사회보험 보수에 포함되고 근로소득세 과세대상으로 처리하는 것이 기본이다. ^fact-conclusion
