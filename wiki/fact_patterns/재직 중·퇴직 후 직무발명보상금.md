# 재직 중·퇴직 후 직무발명보상금

## 대표 지급구조

발명신고·권리승계·출원·등록·실시·처분 등 단계별로 보상규정의 산식에 따라 지급하고, 일부 금액은 종업원이 퇴직한 뒤 지급한다.

## 판정 전 확인할 사실

- 직무발명 해당 여부와 권리 승계·실시권 설정
- 보상 단계, 지급요건, 산식과 사용자 재량
- 근로 제공 대가인지 권리 이전 대가인지
- 지급일과 퇴직일
- 재직 중·퇴직 후 소득 구분
- 연간 비과세 누계

## 영역별 판정

| 영역 | 기본 결론 | 확인 포인트 |
|---|---|---|
| 임금성 | 조건부 | 근로 대가성과 지급의무, 권리 이전 대가 |
| 통상임금 | 일반적으로 신중 | 정기·일률·소정근로 대가성 |
| 평균임금 | 임금성이 인정된 귀속액만 | 발생·지급 시점 |
| 최저임금 | 조건부 또는 제외 | 소정근로 대가성 |
| 근로소득세 | 재직 중·퇴직 후 구분 | 법정 소득 구분과 연 700만원 비과세 한도 |
| 국민연금 | 조건부 | 근로소득 귀속과 지급 시 재직 여부 |
| 건강·장기요양 | 조건부 | 보수 귀속과 퇴직 후 지급 |
| 고용·산재 | 조건부 | 근로소득·보수총액 귀속 |

## 결론 변경 조건

명칭만 직무발명보상금이고 실제로는 일반 성과급인 경우, 사용자 재량 없이 근로성과에 정기 지급하는 경우, 퇴직 후 권리 처분대가로 지급하는 경우는 결론이 서로 달라진다.

## 필수 증빙

발명신고서, 직무발명 확인, 권리승계 통지, 보상규정, 심의자료, 특허·실시·처분 자료, 지급결의서, 퇴직일, 연간 비과세 누계.

## 결론

직무발명보상금은 보상권의 존재만으로 임금성이 정해지지 않는다. 지급 원인·산식·재량·보상 단계·재직 여부를 확정한 뒤 노동법, 소득세와 각 사회보험을 독립적으로 판정한다. ^fact-conclusion
