# 택시 임금 산정과 최저임금 잠탈 쟁점

택시 임금 사건은 같은 운송수입을 두고도 서로 다른 계산축을 가진다. 초과운송수입금이 평균임금에 들어가는지는 근로대가성과 회사의 관리·지배 가능성을 본다. 최저임금 충족 여부는 비교대상 임금과 유효한 소정근로시간을 확정한 뒤 시간급을 계산한다.

따라서 실무 검토는 다음 순서가 안전하다.

1. 운송수입의 귀속과 회사의 확인·관리 가능성을 조사한다.
2. 평균임금 배제합의가 있으면 법정 퇴직금 하한을 계산한다.
3. 소정근로시간 합의가 실제 운행·준비·대기시간과 부합하는지 검토한다.
4. 합의가 무효이면 유효한 종전 규정 또는 보충해석으로 시간을 확정한다.
5. 해당 연도의 최저임금액을 적용해 미달 여부를 계산한다.
