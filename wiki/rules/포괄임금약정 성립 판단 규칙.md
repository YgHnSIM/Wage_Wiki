# 포괄임금약정 성립 판단 규칙

## Issue

급여에 법정수당이 포함되어 있다는 사용자의 주장을 어떤 기준으로 인정하고, 어느 수당까지 약정 범위로 볼 것인가.

## Rule

계약·협약 문언, 임금 항목과 산식, 근로시간·형태와 업무 성질, 협상 경위, 지급 관행과 사업장 실태를 종합하여 법정수당을 정액 임금에 포함하기로 한 객관적 의사합치가 확인되는지를 판단한다. ^claim-rule-inclusive-wage-formation

성립 판단은 ‘포괄임금제’라는 명칭이 있는지보다 해당 임금이 어떤 근로와 수당의 대가로 정해졌는지를 밝히는 계약해석이다. 사용자가 급여에 법정수당이 포함되었다고 주장하는 경우에도 포함 대상과 계산 구조가 객관적으로 확인되어야 한다.

## 판단 순서

1. 근로계약서·단체협약·취업규칙에서 기본임금, 수당의 종류, 예정시간과 산식을 확인한다.
2. 임금대장·임금명세서·급여 이체내역을 통해 문언과 실제 지급이 일치하는지 대조한다.
3. 근로시간표·출퇴근기록·업무기록으로 실제 근로시간의 측정 가능성과 근무형태를 확인한다.
4. 임금협상 자료와 사업장 관행에서 총액 또는 정액수당 외에 추가 수당을 지급하지 않기로 한 의사합치가 있었는지 확인한다.
5. 연장·야간·휴일·연차수당 등 각 수당별로 포함 범위를 따로 확정한다.

## 판단 요소

| 성립을 뒷받침할 수 있는 요소 | 성립을 부정하는 방향의 요소 |
|---|---|
| 수당의 종류와 포함 문언이 구체적임 | 급여를 소정근로의 대가로만 특정함 |
| 총액 또는 정액수당의 산식이 실제 지급과 일치함 | 기본급과 각 수당이 별도 항목·산식으로 지급됨 |
| 협상 과정에서 포함 범위를 합의함 | 계약서가 미기재 사항은 법령에 따른다고 정함 |
| 근무형태상 일정한 초과근로가 예정됨 | 근무시간표·전산기록으로 실제 시간을 쉽게 산정함 |
| 동종 근로자에게 같은 구조가 일관되게 적용됨 | 사용자가 사후적으로만 포괄임금을 주장함 |

어느 요소 하나로 결론을 내리지 않고 전체 사정을 종합한다. 특히 기본급과 법정수당을 구분하여 지급한 사정은 ‘총액 안에 모든 수당이 포함되었다’는 주장과 양립하기 어렵다. ^claim-rule-inclusive-wage-positive-negative

## Exceptions

명시적 약정이 없을 때는 [[묵시적 포괄임금약정 성립 판단 규칙]]의 강화된 기준을 적용한다.

## Conclusion

하나의 계약에서도 연장·야간수당은 포함되지만 휴일·연차수당은 제외되는 등 수당별로 약정 범위가 달라질 수 있다. ^claim-rule-inclusive-wage-scope

## Application

급여명세서에 ‘시간외수당’이 있다는 사실만으로 판단하지 말고 예정시간, 산식, 실제 초과시간 정산을 함께 확인한다. 반대로 계약서에 포괄임금 문언이 있더라도 실제로 기본급과 수당을 별도로 산정·정산했다면 문언만으로 총액형 약정을 인정하지 않는다.

핵심 증거는 근로계약서, 단체협약, 취업규칙, 임금협정, 임금대장, 임금명세서, 근무표, 출퇴근기록, 급여 산출표와 임금협상 자료이다. 문언·산식·지급실무가 충돌하면 실제 지급 목적과 당사자의 객관적 의사를 설명하는 자료를 종합한다. ^claim-rule-inclusive-wage-evidence

## Notes

성립이 인정되어도 적법성이 확정되는 것은 아니다. 약정의 유형, 법정수당 미달 차액, 최저임금, 임금대장·명세서 의무와 2026년 행정감독상 시정 여부는 별도 단계에서 판단한다.
