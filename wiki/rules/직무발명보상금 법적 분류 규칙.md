# 직무발명보상금 법적 분류 규칙

## 판단 규칙

발명진흥법상 정당한 보상권이 존재한다는 사실만으로 근로기준법상 임금성이 자동 확정되지는 않는다. 권리 승계·실시권 설정, 보상규정의 산식과 재량, 근로 제공 대가성, 지급 단계와 재직 여부를 확정한 뒤 노동법·세법·사회보험을 각각 판정한다. ^invention-classification-rule

## 영역별 확인

- 노동법: 근로의 대가인지, 사용자에게 지급의무가 확정되어 있는지 확인한다.
- 세법: 재직 중·퇴직 후 소득 구분과 연 700만원 비과세 한도를 확인한다.
- 사회보험: 해당 소득의 근로소득 귀속과 각 보험의 보수 정의를 적용한다.
- 문서: 발명신고서, 승계통지, 보상규정, 심의자료, 지급결의서와 퇴직일을 보존한다.

## 검토 상태

직무발명보상금 전반의 임금성을 일률적으로 확정할 수 있는 직접 판례를 확보하지 못했으므로 조건부 `review` 상태로 유지한다.
